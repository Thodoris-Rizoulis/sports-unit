# Landing Page Quickstart

**Feature**: Landing Page  
**Date**: 2025-11-23

## Overview

The landing page provides an informational entry point for the Sports Unit platform, showcasing the product to athletes, coaches, and scouts.

## Sections

- **Hero**: Prominent call-to-action with "Login / Register" button
- **Features**: 3 key selling points with icons and descriptions
- **Footer**: Minimal copyright information

## Technical Details

- **Framework**: Next.js with TypeScript
- **Styling**: Tailwind CSS with colors from global.css
- **Components**: shadcn/ui components with Heroicons
- **Responsiveness**: Mobile-first design

## Development

1. Components are located in `/components`
2. Main page at `/app/(landing)/page.tsx`
3. All styling uses Tailwind classes
4. Colors reference global.css variables

## Testing

- Visual verification across breakpoints
- Button interaction states
- TypeScript compilation
